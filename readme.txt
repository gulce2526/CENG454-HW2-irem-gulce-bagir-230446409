# CENG 454 HW1: Sky-High Prototype

**Author:** Gülce
**Role:** Developer

## Description
This is a simple 3D flight simulation prototype built in Unity using the Universal Render Pipeline (URP). 

## Flight Controls
* **Pitch (Up/Down):** Up/Down Arrows or W/S
* **Yaw (Left/Right):** Left/Right Arrows or A/D
* **Roll:** Q and E keys
* **Thrust (Forward):** Spacebar
